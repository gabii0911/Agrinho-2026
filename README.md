# Agrinho-2026
Agroverse - Plataforma Inteligente & Simulador AgrícolaO Agroverse é uma aplicação web desenvolvida para simular e gerenciar a produtividade de uma fazenda virtual, unindo dados estratégicos com mecânicas de jogos de gerenciamento (tycoon). Funcionalidades PrincipaisSetup da Fazenda Virtual: Customização do nome da propriedade.Escolha de Cultivos: Seleção de grãos específicos (ex: Soja).Central de Investimentos: Compra de melhorias tecnológicas.Aumento de Produtividade: Upgrades que impactam diretamente o lucro.Monitor do Simulador: Acompanhamento de métricas em tempo real. Tecnologias UtilizadasFrontend: HTML5, CSS3, JavaScript (ES6+).Estilização: Tailwind CSS / CSS Customizado (Tema Escuro).Ícones: Font Awesome / Lucide Icons. Mecânicas de SimulaçãoA produtividade e a estabilidade da fazenda mudam com base nos investimentos adquiridos na central:Trator: Adiciona +5% de Produtividade.Drone: Adiciona +3% de Produtividade.Irrigação Automatizada: Adiciona +10% de Estabilidade.
